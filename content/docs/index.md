# lanyard.cafe

lanyard.cafe is a webring built for the [lanyard.rest](https://lanyard.rest) community a curated collection of personal sites connected in a ring so visitors can hop from one to the next.

## what is a webring?

A webring is a collection of websites linked together in a circular structure. Each member site includes a small widget with prev / random / next links so visitors can navigate through the ring.
